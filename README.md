## Getting started

This project was build using [clickable](https://clickable-ut.dev/en/latest/)

```Bash
git clone https://gitlab.com/knackwurstking/shift-scheduler.git
cd shift-scheduler
clickable --arch arm64 # or armhf or amd64
```
